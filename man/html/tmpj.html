NAME
       tmpj -  TextMate auxiliary function.

SYNOPSIS
       tmpj [TextMate_project_file]

DESCRIPTION
       tmpj is a function to open a named TextMate project file given just the
       file name prefix.  mdfind does the hunting for you. In  zsh,  it  works
       with the _tmpj completion function.
