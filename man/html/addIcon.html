NAME
       addIcon

SYNOPSIS
       addIcon imagefilename[s]

DESCRIPTION
       addIcon  invokes  /usr/bin/sips  --addIcon  to  put an icon on an image
       file, and can batch process a whole pile of images.

AUTHOR
        William Scott

SEE ALSO
