NAME
       mapcoverdiff -  requires CCP4 to be installed

SYNOPSIS
       mapcoverdiff

DESCRIPTION
       mapcoverdiff  produces  an  Fo-Fc  difference Fourier map that covers a
       specified molecule.  The argument must be one in  which  both  the  mtz
       file and the pdb file share a common prefix (as with the output of ref-
       mac or phenix).

AUTHOR
        William Scott
